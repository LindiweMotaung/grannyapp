# Elderly Care Management App

## Overview

Mobile application for managing an elderly care hostel. Built with Flutter and Dart, backed by Supabase for authentication and data storage.

## Technologies Used

- Flutter
- Dart
- Supabase (authentication & database)
- Provider (state management)

## Features

- Admin dashboard for managing residents, staff, and schedules
- Resident profile management and vital records
- Visitor booking and visit scheduling
- Customer/visitor portal with resident search
- Role-based authentication (admin, visitor, resident)
- Light and dark theme support

## Screenshots

Screenshots are available in the [screenshots/](screenshots/) folder.

## Setup

1. Clone the repository
2. Run `flutter pub get`
3. Create your Supabase project and update `lib/services/supabase_service.dart` with your credentials
4. Run `flutter run`
